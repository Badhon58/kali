# Introduction To Malware

## What is Malware

_Malware_ : Malicious Software Designed to disrupt, damage, or gain unauthorized access to system.

- _Core Characteristic_
  - Executed with malicious intent
  - Can be file-based or file less
  - Often uses obfuscation to evade detection.
  - May communicate with a remote command server (C2)
